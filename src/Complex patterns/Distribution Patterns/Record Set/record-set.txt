Под капотом многих PHP ORM используется PDO (PHP Data Object) который как правило представляет результат запроса в виде индексированного массива ассоциативных массивов.
Вложенные массивы (или также могут быть key-value объекты) это сырые извлеченные данные.
ORM мэпит такие данные к моделям/сущностям.
Также ORM может вернуть такой сырой record set если попросить её сделать это через специальные методы.
