Laravel
    Фреймворк не предлагает использования репозиторий. Предполагается что все необходимое можно сделать через scoup'ы и builder'ы eloquent.
    На практике приходится делать репозитории для сложных выборок, таких например как данные для репортов.
Symfony
    Репозитории один из удобных инструментов фреймворка.
