Если Resource Descriptor это удобная абстракция чего-то иного, то примерами могут быть:
    Active Record - Строка в таблице
    File - отдельный файл
    Excel file components
        Row
        Cell - объект с такими параметрами как цвет, шрифт, размер
    Storage - Хранилище данных / фалов
        Локальное - на месте расположения исполняемого кода
        Удаленное - AWS S3, Google Drive, etc
    Client of specific API - Фасад / Дескриптор удаленных данных
